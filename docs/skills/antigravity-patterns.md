# Skill: Antigravity Patterns (Google)

## Objective
Enable Google's Antigravity agent to interact with Workspace resources and GCP services effectively.

## Instructions for Antigravity
- **Resource Access:** Use the `gas-fakes` patterns to access Sheets, Drive, and Docs from outside the Apps Script IDE.
- **Contextual Search:** When a task involves a Google API, first search the `docs/gcp/` folder for established Solution Architect patterns.
- **Tool Creation:** If Antigravity needs a new "Tool," refer to the `scripts/templates/` for how to wrap GCP APIs or GWS services.

## Common Antigravity Tasks
1. **Analyze Spreadsheet Data:** Retrieve data via `SpreadsheetApp` (emulated) and perform analysis.
2. **Generate Documentation:** Create Google Docs based on code analysis.
