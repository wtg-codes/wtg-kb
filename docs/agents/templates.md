# Agent Templates

Cross-agent compatibility patterns for Jules, Cursor, and more.

## Rule Formats

### .julesrules
Custom rules for the Jules agent.

### .cursorrules
Project-specific instructions for Cursor.

### Antigravity Rules
Format for the Antigravity agent.

## Prompt Templates
- **XML Structure**: Preferred for Claude.
- **Markdown Instruction**: Standard for GPT-based agents.
