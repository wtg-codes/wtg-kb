# Enterprise Security & Google Flow

## Chrome Enterprise Premium (CEP)
CEP provides context-aware access and data protection for the browser.
- **[Ingestion Guide](https://docs.cloud.google.com/chronicle/docs/ingestion/default-parsers/chrome-enterprise-premium):** Collecting CEP context data into Google Security Operations (Chronicle).
- **Security Visibility:** Enhance visibility with GSO and CEP integrations.

## Google Flow
[Google Flow](https://support.google.com/flow/answer/16353333) is an automation platform for Google Workspace.
- **Get Started:** Use Flow to orchestrate tasks across GWS without writing boilerplate Apps Script.
- **Add-on Extension:** Google Workspace Add-ons can extend Flow via the `AddOnsResponseService`.

## PAIR (People + AI Research)
- **[PAIR Guidebook](https://pair.withgoogle.com/guidebook/):** Best practices for designing human-centric AI experiences. Use this when building UI/UX for GWS Add-ons or ADK agents.
