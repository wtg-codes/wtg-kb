# Agent Development Kit (ADK)

The Google Agent Development Kit (ADK) is the foundational framework for building high-performance AI agents.

## SDKs & Toolkits
- **[ADK-Rust](https://github.com/zavora-ai/adk-rust):** High-performance Rust SDK for modular agent components (Models, Tools, Memory, Voice).
- **[ADK-TypeScript](https://adk.dev/get-started/typescript/):** The official TypeScript/Node.js starter for ADK.
- **[adk-rust-mcp-toolkit](https://github.com/zavora-ai/adk-rust-mcp-toolkit):** Production-ready MCP servers built in Rust.

## Resources & Samples
- **[google/adk-samples](https://github.com/google/adk-samples):** Official collection of sample agents.
- **[awesome-adk-agents](https://github.com/Sri-Krishna-V/awesome-adk-agents):** Curated templates and best practices.
- **[awesome-google-adk](https://github.com/tsubasakong/awesome-google-adk):** More curated ADK resources.

## Development Tools
- **[adk-rust-extension](https://github.com/zavora-ai/adk-rust-extension):** VS Code extension for scaffolding and visual building (ADK Studio).
- **[zavora-cli](https://github.com/zavora-ai/zavora-cli):** Rust-based CLI for fast agent iteration.
- **[vllora](https://github.com/vllora/vllora):** Debugging tool for AI agents.
